# 文风提取指南

从轻小说/游戏/小说等源材料中提取写作风格，转化为文风条目（position=2 ↑AT）。

---

## 何时使用本文档

- 用户明确要求"提取文风""总结写作风格""模仿这个作者写作"
- 用户要求转化轻小说并保留原文文风
- **默认的转化任务不读此文档**（除非用户要求）

---

## 提取维度

### 一、写人

#### 1.1 外貌描写
- 原文是白描直叙还是浓墨重彩？
- 常用哪些形容词、感官词？
- 外貌描写是开场集中展示，还是分散在动作中带出？
- 外貌与身份/性格的暗合方式

#### 1.2 动作描写
- 动词选择倾向（精确有力还是模糊留白）
- 节奏：快节奏短句还是慢镜头放大
- 标志性动作的重复使用

#### 1.3 表情与情感传达
- 通过哪些面部器官/身体姿态呈现
- 内心情感是直接叙述还是间接暗示
- 情感词汇的常见选择

### 二、写物

- 物体描写的感官调用（触觉/温度/重量）
- 静态陈列还是动态使用中展示
- 物品与角色的象征关系

### 三、写景

- 写景范围（大场面/角落特写）和视角
- 色彩、光线、声音的敏感度
- 景物与人物情绪的匹配方式
- 时间流动的强调方式

### 四、语言层面

- **句式**：长句/短句/长短交替，平均句长估计
- **用词**：高频词/特色词/偏文还是偏白
- **修辞**：比喻/拟人/排比的频率和类型
- **对话风格**：偏向口语还是书面、对话中的语气词习惯

---

## 输出格式

提取结果应分为两部分：

### 部分一：分析报告（存于 outline.txt）

```
## 文风分析

### 写人手法
- 外貌：<总结>
- 动作：<总结>
- 情感：<总结>

### 写物手法
- <总结>

### 写景手法
- <总结>

### 语言特点
- 句式：<总结 + 平均句长估计>
- 用词：<高频词列表 + 特色词>
- 修辞：<常用修辞类型>
- 对话：<风格总结>
```

### 部分二：文风条目（写入世界书）

文风条目应写入世界书，放在 position=2（↑AT），蓝灯常驻。**内容必须是可操作的指令，不是分析报告。**

```yaml
# 文风条目 content 示例：
作者名称:
作品名称:
文风指引:
  句式:
    - 平均句长: 15-25字/句
    - 短句占比: 60%以上，对话段落用极短句
    - 长句用途: 环境描写和情感段落

  用词:
    - 偏好: 口语化，少用成语，避免文言词汇
    - 高频词: 淡淡/微微/轻轻（用频率高，适度使用避免泛滥）
    - 禁忌: 避免"十分""非常""极其"等程度副词

  写人:
    - 外貌: 分散在动作中带出，不集中描写
    - 动作: 精确动词，一句话一个动作，不堆砌
    - 情感: 90%通过动作和对话间接表达，不写内心独白

  写景:
    - 通过角色视角写景，不用全知视角
    - 景物承载情绪，但不过度渲染
    - 色彩用词克制，以青/白/灰等冷色调为主

  写物:
    - 在角色使用物品时描写，不单独陈列
    - 关注触感（质地/温度/重量）多于外观

  对话:
    - 口语化，带轻微方言语气词（如"啦""嘛"）
    - 对话和动作交叉写，避免连续3句以上的纯对白
```

注意：文风条目的 content 是**指令格式**，不是分析报告格式。每条指令 AI 可以直接执行。

分析报告（部分一）存入 outline.txt，文风条目（部分二）写入世界书。
